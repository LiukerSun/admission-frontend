## ADDED Requirements

### Requirement: Login page provides navigation to home page
The system SHALL display a link on the login page that navigates the user to the home page (`/`).

#### Scenario: User clicks home link on login page
- **WHEN** a user is on the login page and clicks the link to return to the home page
- **THEN** the system navigates the user to the home page (`/`)

### Requirement: Register page provides navigation to home page

#### Scenario: User clicks home link on register page
