## Context

当前登录页面 (`/login`) 和注册页面 (`/register`) 使用 Ant Design 的 `Card` 组件渲染认证表单。页面底部仅提供注册/登录互转的链接，缺少返回首页 (`/`) 的导航入口。用户若临时改变主意或误进入认证页面，没有便捷的退出路径。

## Goals / Non-Goals

**Goals:**
- 在登录和注册页面中提供显式的返回首页入口
- 保持与现有页面视觉风格一致
- 使用最小改动实现，不引入新依赖

**Non-Goals:**
- 不修改认证逻辑、表单验证或 API 调用
- 不重新设计整个认证页面布局
- 不涉及导航栏或全局布局的改动

## Decisions

- **方案：在 Card 底部增加文字链接**
  - 使用 Ant Design 的 `Link` 组件（来自 `react-router-dom`）跳转到首页 `/`
  - 放在表单提交按钮下方、与现有的"还没有账号？立即注册"链接同一区域
  - 理由：改动最小，与现有 UI 风格一致，用户容易发现

## Risks / Trade-offs

- 无显著风险。这是纯 UI 增强，不影响功能逻辑。
